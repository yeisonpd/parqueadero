# 🚗 Sistema de Control de Parqueadero

Aplicación web desarrollada con **Node.js** y **Express** para gestionar la **entrada y salida de vehículos** en un parqueadero.

El sistema permite registrar vehículos, controlar su ingreso y salida, y consultar el historial de registros.

---

# 📋 Requisitos

Antes de ejecutar el proyecto asegúrate de tener instalado:

* Node.js
* npm (gestor de paquetes de Node)

Puedes verificarlo con:

```
node -v
npm -v
```

---

# ⚙️ Instalación

1. Clonar el repositorio

```
git clone https://github.com/yeisonpd/parqueadero.git
```

2. Entrar a la carpeta del proyecto

```
cd parqueadero
```

3. Instalar las dependencias

```
npm install
npm install express
```

---

# ▶️ Ejecutar la aplicación

Para iniciar el servidor ejecuta:
antes de ejcutar la aplicaion ve a db.js y agregar la contraseña de tu base de datos(mySQL) en password
```
node server.js
```


# 🌐 Acceder a la aplicación

Una vez iniciado el servidor, abre tu navegador en:

```
http://localhost:3000
```
